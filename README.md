# 🏫 Junior School Management System

A comprehensive, premium web application designed for junior schools to manage their online presence, admissions, and student facilities.

## 🚀 Live Preview (Local)
To view the site on your computer:
1. Ensure **XAMPP** is running (Apache & MySQL).
2. Visit: [http://localhost:8080/junior_school](http://localhost:8080/junior_school)

## ✨ Key Features
- **Admin Dashboard**: Full control over school content and settings.
- **Student Registration**: Easy-to-use admission forms for new students.
- **Gallery & Facilities**: Beautifully presented school facilities and event photos.
- **Staff Directory**: Detailed profiles of the teaching and administrative team.
- **Events & Vacancies**: Real-time updates on school activities and career opportunities.
- **SEO Optimized**: Fully integrated sitemap and robots.txt for better search visibility.

## 🛠️ Technology Stack
- **Backend**: PHP 8.x
- **Database**: MySQL (MariaDB)
- **Frontend**: HTML5, Vanilla CSS3, Javascript
- **Icons/Styling**: Responsive assets and custom fonts.

## 📦 Installation
1. Clone this repository into your `xampp/htdocs` folder.
2. Import the `database.sql` file into your phpMyAdmin.
3. Configure `config.php` with your database credentials.
4. Access the site via your local server URL.
